Extensão para Navegadores para Visualização dos Produtos
Para que seja possível você visualizar os produtos nas consultas, é importante que você instale uma extensão no seu navegador. Essa extensão está disponível para Google Chrome, Mozilla Firefox e Microsoft Edge e é simples de se instalar e utilizar. Após instalar a extensão, coloque-a no modo ON, e você estará pronto para poder realizar as consultas de produtos na aplicação.
